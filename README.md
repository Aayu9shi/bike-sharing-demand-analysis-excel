# Bike Sharing Demand Analysis Using Microsoft Excel

## Project Overview

This project presents a comprehensive analysis of bike-sharing demand using Microsoft Excel. The objective of the project is to analyze historical bike rental data and identify the key factors influencing customer demand. The study focuses on deriving business insights through structured data analysis and visualization techniques.

The analysis was performed using advanced Excel functionalities such as Pivot Tables, Pivot Charts, Conditional Formatting, Data Transformation, Trend Analysis, and Statistical Summaries.

This project demonstrates how Excel can be effectively used for solving real-world business problems and supporting data-driven decision-making.

---

# Business Problem

Bike-sharing companies operate in a dynamic environment where customer demand fluctuates based on multiple factors such as:

- Weather conditions
- Seasons
- Temperature
- Working days
- Time of day
- Customer type

Improper demand forecasting can lead to:

- Bike shortages during peak hours
- Underutilization during low-demand periods
- Operational inefficiencies
- Reduced customer satisfaction

The purpose of this analysis is to uncover demand patterns and generate actionable business recommendations that can improve operational efficiency and customer experience.

The major objectives of this project are:

# Key Insights

The analysis generated several important business insights.

## Major Findings

- Bike rental demand increased significantly over time
- Summer and Fall recorded the highest demand
- Winter season showed the lowest rental activity
- Clear weather conditions resulted in maximum rentals
- Registered users contributed the majority of rentals
- Peak demand occurred during office commuting hours
- Temperature positively influenced rental demand
- High humidity and windspeed negatively impacted usage
